getAllPRs

Grabs all PRs in repository and filters them based on those associated with a contributor user status and calculates metric percentage using, "total number of merged contributor PR / total number of contributor PRs"

getAllPRs(owner: string, libName: string, contributors: object): float
Parameters
owner (string) representing the owner of the repository
libName (string) representing the name of the repository/library
contributors (object) with keys as contributor login names, and values as number of commits associated with user
Returns
float: percentage showing the % of merged contributors PRs in a library
Example
owner=google libName=gson contributors={user1:12,user2:3}
